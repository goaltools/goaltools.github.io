Meet Goal Tools A set of Open-Source tools for high productivity web-development in Go language

Code Generation

Goal Tools rely on the concept of code generation rather than runtime reflection. As a result better type safety and productivity of the end developer are provided. Catch errors when you are building not running your app. And focus on things that matter, Goal will take care of the rest.

Customizability

Components of Goal Tools are built with compatibility with the standard library in mind. Plus they are loosely coupled and thus can be easily replaced with alternative implementations. The result is you can use your own router, template system, reverse routes generator or any other component.
